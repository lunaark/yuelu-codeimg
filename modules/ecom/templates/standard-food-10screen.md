# 食品饮料标准版模板（10屏）

基于 NFC 橙汁详情页实战验证的模板结构。

## 配色 Token

| 角色 | 色值 | 用途 |
|------|------|------|
| 主色 | #FF6D00 | 标题栏、按钮、强调 |
| 辅色 | #FF8C00 | 图标、装饰、浅标注 |
| 浅橙 | #FFE0B2 | 卡片边框、分隔 |
| 背景 | #FFFAF3 | 主背景色（米白暖调） |
| 浅底 | #FFF5E6 | 图片占位区、次级卡片 |
| 正文黑 | #2B2B2B | 主标题 |
| 正文灰 | #666 / #888 | 副标题、说明文字 |

## 10 屏结构

### Screen 1 — 首屏（1080×1440）
- 产品主图居中（大占位 600×700）
- 核心卖点大字（如"0 水 0 糖 0 添加"）
- 品名 + 规格
- 价格锚点（划线价 + 现价）
- 底部装饰条

### Screen 2 — 痛点对比（1080×1440）
- 标题："你喝的真的是果汁吗？"
- 左右对比布局：
  - 左：竞品/旧方案（灰调、❌ 标记）
  - 右：本产品（主色调、✅ 标记）
- 底部一句话总结

### Screen 3 — 四大卖点（1080×1920，内容多给高度）
- 标题："四大理由选择 XX"
- 2×2 大卡片布局（每个 480×400）
- 每卡片：图标区 + 主标 + 2 行说明
- 底部辅助信息条

### Screen 4 — 工艺/流程（1080×1440）
- 标题："从 XX 到 XX 只需 N 步"
- 横向 4 步流程：圆形图标 + 步骤名 + 说明
- 步骤间用箭头/连接线
- 底部强调文字

### Screen 5 — 口碑评价（1080×1440）
- 标题："真实用户这样说"
- 3 张评价卡片纵向排列
- 每卡片：头像占位 + 昵称 + 星级 + 评价文字
- 底部统计（如"好评率 99%"）

### Screen 6 — 配料/成分（1080×1440）
- 标题："配料 & 营养 · 透明安心"
- 左右分栏：
  - 左：配料大字（如"100% 鲜脐橙"）
  - 右：营养亮点列表（图标 + 文字）
- 底部"不含清单"（❌ 列表）

### Screen 7 — 使用场景（1080×1440）
- 标题："随时随地，XX"
- 2×2 场景卡片
- 每卡片：场景图占位 + 场景名 + 2 行描述
- 底部小贴士

### Screen 8 — 产品规格（1080×1440）
- 标题："产品规格详情"
- 产品正面图占位
- 左右分栏规格表：
  - 左：参数表（净含量、规格、价格、储存…）
  - 右：包装优势列表
- 底部包装展示占位

### Screen 9 — FAQ（1080×1920，Q&A 多给高度）
- 标题："买家关心 FAQ"
- 5 个 Q&A 卡片纵向排列
- 每卡片：橙色标题栏（Q标题）+ 白色回答区
- 常见问题参考：
  1. 产品是什么 / 区别
  2. 添加剂问题
  3. 适用人群
  4. 外观/质地疑虑
  5. 保存/使用方法

### Screen 10 — CTA 下单（1080×1080，方形收尾）
- 产品图居中（圆形虚线框）
- 价格 + 品名大字
- 副标语（核心利益点）
- 大号 CTA 按钮（"点击立即下单"）
- 底部：物流承诺 + 售后保障
- 装饰条

**⚠️ 重要：Screen 10 顶部背景色必须从 #FFFAF3 渐变开始，与 Screen 9 底部衔接。**

## SVG 线框约定

- 所有占位区用 `stroke-dasharray` 虚线框标记
- 图片占位区内写文字说明（如"产品主图"、"早餐场景图"）
- 文字用实际文案，不用 Lorem ipsum
- 背景色用目标配色，方便 GPT 理解风格意图
